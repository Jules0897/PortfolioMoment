Vaardigheden

Talen die ik ken

Momenteel heb ik gewerkt met de talen HTML, CSS, JavaScript, PHP, SQL, C# en Python. Ik ken natuurlijk niet alle talen even goed dus heb ik onder deze tekst een tabel gemaakt waar ik mezelf beoordel per taal. Verder heb ik ook gewerkt met tools zoals Bootstrap, SASS en TypeScript.

Zelf werk ik het liefst met talen zoals HTML, CSS, PHP en C#. Deze talen heb ik op school het meeste mee gewerkt waardoor ik ook de meeste ervaringen heb met deze talen, SQL vind ik ook leuk om mee te werken maar ik kan het opzetten van de databases ooit wat minder leuk vinden. JavaScript kan ik goed mee overweg maar dit kost me vaak wat tijd om weer even in te komen omdat ik deze taal niet vaak gebruik. Hoewel ik Python snap heb ik er niet veel mee gewerkt. We hebben op school wat korte Python opdrachten gehad maar ik vond dat die niet veel voorstelde.
